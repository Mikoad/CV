Qui suis-je

Je m'appelle Adrien Mikolajczyk et je suis un développeur web front-end junior passionné et créatif. J'ai récemment validé mon titre professionnel de développement web et web mobile et je suis actuellement à la recherche d'une alternance en développpement web. Je suis enthousiaste à l'idée de contribuer à des projets stimulants et d'apprendre de nouvelles technologies. Je m'assure que les sites que je développe sont accessibles, réactifs et optimisés.

Développement

Je maîtrise les langages clés du développement web tels que PHP, HTML, CSS et JavaScript. Au cours de cette formation intensive de 6 mois, j'ai acquis une solide compréhension de ces langages. J'approfondie mes compétences en Javascript et je découvre le framework React en me formant en autodidacte.

Outils

Je maîtrise également les outils suivants : Figma, Bootstrap et Github. Figma est mon outil pour l'UI/UX, Github pour la gestion de versions et le travail en équipe. Je maîtrise également Make pour automatiser diverses choses (notifications, mail automatique, base de données). Je suis capable de réaliser des projets web de qualité en utilisant tous ces outils.
